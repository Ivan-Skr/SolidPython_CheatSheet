linear_extrude(heigh; center; slices; twist) (2D_figure)

Linear Extrusion is an operation that takes a 2D object as input and generates a 3D object as a result. Twist is the number of degrees the shape is extruded. Setting the parameter twist = 360 extrudes one turn. The direction of rotation is clockwise. If center is false the linear extrusion Z range is from 0 to height; if it is true, the range is from -height/2 to height/2. Scales the 2D shape by this value over the height of the extrusion. Slices like segments.
