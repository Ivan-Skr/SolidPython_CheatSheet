Converts image file to figure. Currently only PNG images are supported. the height for the pixel is determined by converting the color value to Grayscale using the linear luminance for the sRGB color space (Y = 0.2126R + 0.7152G + 0.0722B). The gray scale values are scaled to be in the range 0 to 100.

surface(file = "file.png", center = True/False, invert = True/False)
