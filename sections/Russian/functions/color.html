color("color"; alpha) (figure)

Дочерние элементы отображаются с использованием указанного значения RGB color + alpha. Это используется только для предварительного просмотра F5, поскольку CGAL и STL (F6) в настоящее время не поддерживают color. Значение альфа по умолчанию равно 1.0 (непрозрачный), если не указано иное (от 0 до 1).

color(c = "color", alpha = x) (figure)
