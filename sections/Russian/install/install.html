Руководство по установке SolidPython, если вашей операционной системой является Windows

1. Нажмите Win+R

2. Напишите "cmd"

3. Нажмите "OK"

4. Напишите "pip install solidpython"

5. Нажмите "Enter"

Если вы видите "Successfully installed solidpython", значит, установка прошла успешно
